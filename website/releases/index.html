Version 1.0.0 Latest
January 15, 2024

We're excited to announce the first public release of Jubilee Browser! This release includes all core features for browsing, tab management, and the Jubilee Bibles integration.

Features

  • Dual-mode browsing: Switch between Internet and Jubilee Bibles modes
  • Tab management with drag-and-drop support
  • Native inspire:// protocol handler
  • Automatic updates with differential downloads
  • Built-in content filtering and blacklist support
  • Session restoration across updates
  • Bookmark and history management
  • Dark mode support (follows system preference)

Enterprise Features

  • Silent installation with command-line options
  • Per-user and per-machine installation modes
  • Group Policy compatible configuration
  • Pre-configuration with JSON files
  • Detailed installation logging

Security

  • Process isolation between main and renderer
  • Session partitioning between browsing modes
  • Content Security Policy enforcement
  • Automatic update verification with code signing
